# Contributing

Good additions for this kit should help someone trace a DSA pattern faster inside Obsidian Excalidraw.

## Script guidelines

- Keep scripts dependency-free.
- Prefer readable code over clever code.
- Use semantic labels: `current`, `visited`, `window`, `answer`, etc.
- Avoid network requests.
- Avoid editing files outside the active Excalidraw canvas.
- Keep generated diagrams fully editable.

## Useful future scripts

- Heap / priority queue visualizer
- Trie visualizer
- Union-Find parent/rank visualizer
- Recursion tree generator
- Sliding-window timeline generator
- DP transition arrows
