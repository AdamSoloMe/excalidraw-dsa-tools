# NOTICE

This project is an improved MIT-compatible derivative/starter kit inspired by:

- Repository: https://github.com/anthonynuge/excalidraw-dsa-kit
- Original author: Anthony Nguyen
- Original license: MIT

The original project provides Obsidian Excalidraw scripts for parsing DSA inputs,
creating blank data structures, adding pointers, toggling state colors, and resetting state.

This version keeps the same educational goal but reorganizes and expands the scripts with:

- additional data structures
- safer input parsing
- better DSA tracing helpers
- semantic state colors
- trace cards for algorithm notes

The original MIT notice is preserved in `LICENSE`.
